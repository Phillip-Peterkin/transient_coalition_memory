# Empty curriculum — learning under emptiness

## Council of six (#2)

| Agent | Lane | Injected into the design |
|---|---|---|
| 1 | Emptiness as sensory channel | Absence is structured observation, not missing data |
| 2 | ACI/sheath gap | Empty → `null_pe` updated, sheath used to no-op; fix with two-phase lesson |
| 3 | Tacitheca maturation | Silence bits distinguish *kinds* of absence; 1→12 meanings |
| 4 | World principles | Emptiness as second curriculum; no new APIs |
| 5 | Anti-theft | **REJECT** separate VacancyTutor class — fold into Mnemosheath empty path only |
| 6 | Code shape | `prime_absence` → pending → complete on truth (change vs stay) |

## Mechanism

1. **Phase 1 (predict, no truth):** if `empty` / `low_delta`, stash pending lesson
   and increment silence-bit `primed` counters. Track `time_since_evidence`.
2. **Phase 2 (feedback):** if `majority_vote is None`, teach silence bits whether
   emptiness preceded **CHANGE** vs **STAY**.
3. Silence-bit `change_rate` joins the context vote (raises urgency when vacancy
   historically foreshadows flips).
4. Agreement curriculum unchanged for non-empty batches.
